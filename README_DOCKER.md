# CheckCheck

A collaborative, self-hostable, offline-capable checklist application (web UI + REST API).

**Status: Pre-Alpha.**

## Tags

- `latest` — newest stable release
- `beta` — newest pre-release
- `dev` — latest build from `main` (bleeding edge)
- `<version>` — a specific release (e.g. `0.1.0`)
